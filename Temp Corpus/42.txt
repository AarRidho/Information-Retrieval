Abstract class merupakan sebuah class yang pada pendeklrasiannya ditambahkan keyword abstract. Hal ini dilakukan untuk membedakan class yang bersifat abstract dengan class biasa. Abstract Class memiliki sifat yakni dapat mendefinisikan Abstract Methodnya bisa juga tidak di deklarasikan. Selain itu Abstract Class ini tidak dapat di instansiasi akan tetapi dapat di inherit oleh subclassnya.
